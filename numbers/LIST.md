# fdb.tkoz.me production list

- Fibonacci
  - $ F_0=0, F_1=1, F_n=F_{n-1}+F_{n-2} $
- Lucas
  - $ L_0=2, L_1=1, L_n=L_{n-1}+L_{n-2} $
- Repunits
  - bases $ 2 \leq b \leq 20 $
  - $n$-th term is $ (b^n-1)/(b-1) $ where $ n \geq 0 $
